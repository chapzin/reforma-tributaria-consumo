# 20. Matriz de decisão: permanecer, hibridizar ou migrar

## Quatro cenários, não dois

1. permanecer no Simples com IBS/CBS no DAS;
2. permanecer no Simples e optar pelo regular de IBS/CBS;
3. migrar para Lucro Presumido;
4. migrar para Lucro Real.

O terceiro e quarto mudam IRPJ, CSLL, folha, obrigações e governança; não são mera extensão do cálculo de IBS/CBS.

## Modelo de comparação

Para cada cenário e mês:

> resultado após tributos = receita sem IBS/CBS − custos sem créditos recuperáveis − tributos não recuperáveis − custo de conformidade − custo financeiro

Em seguida, simule o preço necessário para preservar margem e o custo líquido percebido pelo cliente.

## Matriz ponderada

Pontue de 1 (desfavorável) a 5 (favorável):

| Critério | Peso sugerido | DAS | Híbrido | Presumido | Real |
|---|---:|---:|---:|---:|---:|
| carga total | 30% |  |  |  |  |
| crédito para cliente | 20% |  |  |  |  |
| crédito de compras | 15% |  |  |  |  |
| custo operacional | 15% |  |  |  |  |
| caixa | 10% |  |  |  |  |
| flexibilidade/risco | 10% |  |  |  |  |

Pesos devem refletir a estratégia. Em B2B, crédito do cliente pode valer mais; em varejo de baixa margem, caixa e custo operacional podem dominar.

## Quando o Simples tradicional pode ser desvantajoso

- clientes regulares exigem crédito e comparam custo líquido;
- concorrentes transferem crédito maior;
- compras tributadas seriam fortemente creditáveis;
- a empresa aceita maturidade operacional do regular;
- o desconto comercial para neutralizar crédito perdido supera a economia tributária.

## Quando o híbrido pode ser desvantajoso

- base de custo é majoritariamente folha;
- clientela é B2C;
- pouca compra gera crédito;
- regime regular pressiona margem/caixa;
- ERP e controles não sustentam crédito e eventos;
- benefício de simplicidade é crítico à operação.

## Reorganização societária

Separar canais, atividades ou empresas pode ser legítimo quando há autonomia operacional, razões comerciais, pessoas, ativos e riscos próprios. Fracionar artificialmente receita para permanecer no limite, simular independência ou deslocar contratos sem substância cria risco tributário, societário e trabalhista.

## Teste de sensibilidade

Rode pelo menos:

- receita −10%, base e +20%;
- B2B em 40%, 70% e 90%;
- alíquota-padrão em três hipóteses;
- crédito de compras conservador, base e otimista;
- inadimplência e prazo de crédito;
- perda de 1, 3 e 5 pontos de margem;
- custo mensal de conformidade.

Uma opção só é robusta se continua razoável em mais de um cenário.

## Memorando de decisão

Registre dados, premissas, fonte, cenários descartados, riscos, data de reavaliação e aprovadores. Isso transforma “acho que compensa” em governança consultiva.
